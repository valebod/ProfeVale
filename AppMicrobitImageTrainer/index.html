❓ Preguntas Frecuentes

¿Dónde se guardan las imágenes que uso para entrenar?
Las imágenes que capturas o subes para entrenar el modelo no se almacenan ni se envían fuera de tu dispositivo. Todo el procesamiento ocurre localmente en tu navegador, garantizando privacidad y seguridad.

¿Puedo descargar y volver a subir mi modelo?
Sí, puedes descargar el modelo entrenado y volver a subirlo más tarde para continuar tu trabajo o compartirlo con otros.

¿Puedo usar mi modelo en la app de Teachable Machine?
Sí, puedes copiar el link compatible y usarlo directamente en la app de Teachable Machine de este sitio.
